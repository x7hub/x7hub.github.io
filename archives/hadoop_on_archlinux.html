hadoop on archlinux

Fri 21 June 2013
By x7

搬迁自以前的新浪博客～

参考：

Hadoop - ArchWiki

Single Node Setup

  • 安装：
yaourt hadoop
  • 主要目录是这个：
/etc/hadoop/
  • 当然配置文件还是在这里。
/etc/hadoop/
  • 检查java环境配置。
echo $JAVA_HOME

  • 安装时生成了rsa密钥一对，还需要将本地用户的公钥添加到/etc/hadoop/.ssh/authorized_keys,用于ssh验证。

  • 格式化新的分布式文件系统

hadoop namenode -format
  • 启动服务,需要防火墙打开9000和9001端口，即使单机hadoop也需要，不明
bin/start-all.sh

或者启动systemd的一排service

  • 启动后运行example测试，首先把测试用的文件写到hadoop fs
hadoop fs -put /etc/hadoop input
  • 运行测试
hadoop jar /usr/lib/hadoop/hadoop-examples-*.jar grep input output 'dfs[a-z.]+'
  • 查看结果
hadoop fs -get output output
cat output/*
